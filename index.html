Find My Cite

Are you a Zotero user? Do you ever wish you could search your libraries by ideasâ€”not keywords? Then this page is for you. Follow the steps below to download a information from one of your group libraries and search it using an "idea" search. That is, it will match your search with the semantic content of text from your library. That means you don't have to use the exact language found in a text to find a cite.

To make this site widely available (i.e., let me run it from a free web server), the work is offloaded to your browser. This means you'll have to reload a bunch of stuff every time you load this page. So you might consider leaving this tab open. ;)

API Key

In order to use this tool, you will need to generate a Zotero API key and provide that key with read access to a group library. Just follow these directions and paste the API key in the field below. Be sure to write down the key and to provide it with READ access to your group or groups.

Group ID

This tool works by accessing the contents of a group library. If you don't have one already, you will need to create a group and fill it with content. We perform our search on the full text content of your group's library. If it isn't part of your library, we can't search it. Once you have a group, cut and paste its ID into the field below. The ID is the string of numbers in the URL between /groups/ and your group's name. For example, 2305186 is the ID found in the following url: https://www.zotero.org/groups/2305186/library_name/.

Your Search

This search will do its best to match the semantic content of the text you input with a sentence in your library. For best results, write your query/proposition as a full sentence. Because the work is being done by your browser, and I didn't want to overload it, this tool only knows 25,000 words. So, the search can be slow for big libraries, and matches aren't perfect. That being said, this search beats a simple "find this phrase" or "find these words" search, and it's a free tool. ;)
